## Overview

DataViz Engine is a TypeScript library for rendering large-scale datasets in 3D using WebGL instanced rendering and custom GLSL shaders. It targets analytical dashboards where standard chart libraries break down at scale — rendering 1M+ points at 60fps.

## Motivation

Standard charting libraries (D3, Chart.js, Plotly) begin to struggle past ~100K data points. GPU-accelerated WebGL can render orders of magnitude more, but writing shaders from scratch for each use case is impractical. DataViz Engine provides a high-level API with low-level performance.

## Architecture

### Rendering

- **Instanced mesh rendering** via `THREE.InstancedMesh` — single draw call for millions of points
- **Custom vertex shaders** for per-point color mapping, size encoding, and animation
- **LOD system**: coarsens geometry at camera distances to maintain frame budget

### Data Pipeline

- Typed `Float32Array` input buffers for zero-copy data transfer to GPU
- Streaming ingestion via Web Workers — UI thread never blocks
- Spatial index (k-d tree) for selection and hover queries

### Interaction

- Raycasting against instanced geometry in a Web Worker
- Selection, filtering, and highlight states managed in shader uniforms
- Camera controls: orbit, fly, first-person

## API

```typescript
const viz = new DataVizEngine({ canvas, theme: 'dark' });

viz.addLayer('scatter3d', {
  data: myFloat32Buffer,
  colorMap: 'plasma',
  sizeField: 'value',
});

viz.render();
```
