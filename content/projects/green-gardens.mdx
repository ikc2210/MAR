## Overview

Green Gardens is an IoT sensor network for measuring and visualizing carbon sequestration in urban community gardens. Raspberry Pi nodes equipped with custom sensor arrays measure CO₂ levels, soil conductivity, humidity, and temperature. Data flows to a central dashboard enabling gardeners to optimize for carbon capture.

## Background

Urban gardens sequester meaningful amounts of CO₂, but there was no affordable way for community gardeners to measure or demonstrate this impact. Green Gardens was designed to make that data visible and actionable — and to support the provisional patent application filed in 2020.

## Hardware Design

Each sensor node consists of:

- **Raspberry Pi Zero W** as the compute unit
- **SCD30 NDIR CO₂ sensor** for atmospheric carbon measurement
- **Capacitive soil moisture + conductivity probe** (custom PCB)
- **DHT22** temperature/humidity sensor
- **18650 LiPo battery pack** with 3W solar charging

The custom PCB was designed in KiCad and fab'd by JLCPCB. Nodes run on a single charge for 5–7 days outdoors.

## Software Stack

- **Node firmware**: Python + CircuitPython libraries, MQTT publish over Wi-Fi
- **Backend**: MQTT broker (Mosquitto) → InfluxDB time-series storage
- **Dashboard**: Grafana with custom panel plugins showing cumulative CO₂ absorbed per garden plot

## Patent

Filed US Provisional Patent (Application No. filed Apr 2020) for the multi-modal carbon sequestration monitoring device. Patent covers the combined sensor array design and the per-plot attribution algorithm.

## Impact

- 6 sensor nodes deployed across 3 community gardens in Irvine, CA
- Measured 47 kg CO₂ sequestered over 18 months across sites
- Data used in a grant proposal that secured $12K for garden expansion
